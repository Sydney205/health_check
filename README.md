# Flask + Tailwind project skeleton

This is a <strong>Flask</strong> project template (Skeleton) with <strong>Tailwindcss</strong> for creating modern design and beautiful interfaces. Generated by SkelPro [SkelPro](https://github.com/SkelPro)

## Setup
The best way to setup this template is to scaffold this template using `skelpro`:

```sh
skelpro create https://raw.githubusercontent.com/Sydney205/Skeletons/refs/heads/main/flask-tailwind.skel.json
```
You can also download the raw file and scaffold it locally using skelpro.

- ### Install dependencies
    This project uses both `pip` and `npm`.<br>

    To install the Python dependencies used in this project run:
    ```sh
    pip install -r requirements.txt
    ```

    and then run `npm install` to install Nodejs dependencies:

    ```sh
    npm install
    ```

    ### Development
    After installation process is completed , there are remaining steps to start developing:

    To activate `Tailwindcss` run:
    ```sh
    npm run tailwind
    ```

    Start flask development server:
    ```sh
    python run.py
    ```
And congratulations you have successfully setup the Flask + Tailwindcss project.
